Detailed Description

DAWN Logger adapter.

To make use of this Logger class, you are required to register your own implementation of a Logger via registerLogger. The registered Logger has to implement the LoggerInterface. By default no Logger is registered and no logging is performed.

The following snippet can be seen as a minimal working example:

#include <dawn/Support/Logging.h>
#include <iostream>
class MyLogger : dawn::LoggerInterface {
public:
void log(LoggingLevel level, const std::string& message, const char* file, int line) {
std::cout << file << ":" << line << " " << message << std::endl;
};
int main() {
myLogger = new MyLogger;
dawn::Logger::getSingleton().registerLogger(myLogger);
DAWN_LOG(INFO) << "Hello world!";
delete myLogger;
}
