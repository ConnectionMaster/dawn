Detailed Description

A Stage is represented by a collection of statements grouped into DoMethod of non-overlapping vertical intervals.

A Stage usually corresponds to a inner loop nest (usually an ij-loop). In CUDA gridtools stages are separated by a __syncthreads() call in a kernel.

bool dawn::Stage::overlaps ( const Stage &  other) const

Check if the fields of the other stage overlap with the fields of this stage in the vertical interval (incorporating the vertical extents of the fields)

Note that this is more accurate than

bool stagesOverlap = this->getExtendedInterval().overlaps(other.getExtendedInterval());

as the check is performed on each individual field.
