enum dawn::Cache::CacheIOPolicy

IO policies of the cache.

Enumerator
unknown 

Not yet set.

fill_and_flush 

Read values from the cached field and write the result back.

fill 

Read values form the cached field but do not write back.

flush 

Write values back the the cached field but do not read in.

epflush 

End point cache flush: indicates a flush only at the end point of the interval being cached

bpfill 

Begin point cache fill: indicates a fill only at the begin point of the interval being cached

local 

Local only cache, neither read nor write the the cached field.

enum dawn::Cache::CacheTypeKind

Available cache types.

Enumerator
IJ 

IJ caches require synchronization capabilities, as different (i,j) grid points are processed by parallel cores. GPU backend keeps them in shared memory

K 

Processing of all the K elements is done by same thread, so resources for K caches can be private and do not require synchronization. GPU backend uses registers.

IJK 

IJK caches is an extension to 3rd dimension of IJ caches. GPU backend uses shared memory

bypass 

bypass the cache for read only parameters
