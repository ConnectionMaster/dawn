Detailed Description

Position of a stage.

The position first identifies the multi-stage (MultiStageIndex) and afterwards the stage within the multi-stage (StageOffset). A StageOffset of -1 means before the first stage in the multi-stage.

Example: 

+- MultiStage0-+
|              | <------ Position(0, -1)
| +----------+ |
| |  Stage0  | | <-----  Position(0, 0)
| +----------+ |
|              |
| +----------+ |
| |  Stage1  | | <-----  Position(0, 1)
| +----------+ |
+--------------+

+- MultiStage1-+
|              | <------ Position(1, -1)
| +----------+ |
| |  Stage0  | | <------ Position(1, 0)
| +----------+ |
+--------------+
