Detailed Description

Abstract base class of all optimization and analyzer passes.

To create a new Pass proceed as follows:

1) Inherit from Pass and implement the Pass::run method. Remeber to return true on success! 2) Register your new Pass in the DAWNCompiler::compile method in DAWNCompiler.cpp at the position you would like it run.
